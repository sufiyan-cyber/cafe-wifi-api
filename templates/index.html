☕ Cafe & Wifi API

A RESTful API service for discovering and managing cafes.

Available Endpoints

GET /all → List all cafes
GET /random → Random cafe suggestion
GET /search?loc=city → Search cafes by location
PATCH /update-price?id=&new_price= → Update coffee price
DELETE /report-closed?id=&api_key= → Delete cafe
Interactive API Documentation

You can test every endpoint directly from the live documentation interface.

Open API Documentation
Built using Python Flask • SQLAlchemy • Hosted on Render
